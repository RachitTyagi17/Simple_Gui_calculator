Simple GUI Calculator

Description

This project is a simple calculator built with Python and Tkinter. It supports basic arithmetic operations like addition, subtraction, multiplication, and division.

How to Run the Project

  1. Clone the repository: git clone https://github.com/RachitTyagi17/Simple_Gui_calculator.git
  2. Navigate to the project directory: cd Simple_Gui_calculator
  3. Run the script: python calculator.py
